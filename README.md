## Corporate DevOps CI/CD Project

This project demonstrates an end-to-end DevOps pipeline using:
- Jenkins
- Maven
- Docker
- Kubernetes
- SonarQube
- Trivy
- Prometheus & Grafana

### Flow
1. Developer pushes code to Git
2. Jenkins triggers pipeline
3. Maven builds application
4. SonarQube performs code quality checks
5. Docker image is built
6. Trivy scans image for vulnerabilities
7. Application is deployed to Kubernetes
8. Prometheus & Grafana monitor the cluster
